Computer Vision Behavior Analysis Risk Decision

AI Safety Detection Team Project

영상 데이터를 기반으로 정상/이상 행동을 분석하고, 위험 물체 탐지 결과를 함께 활용해 상황별 위험도를 판단하는 AI 안전 시스템 팀 프로젝트입니다. 최종적으로 위험 점수에 따라 알림, 방송, 신고와 같은 대응 흐름까지 연결하는 것을 목표로 개발하고 있습니다.

팀 프로젝트 진행 중
Problem 위험 상황을 놓칠 수 있음

쓰러짐, 이상행동, 위험 물체 등은 사람이 직접 확인하지 않으면 즉시 대응이 어려운 문제가 있습니다.

Solution 영상 기반 AI 판단

행동 분석 모델과 객체 탐지 모델을 결합해 상황을 분석하고 위험 점수를 계산합니다.

Goal 자동 대응 흐름 구현

위험도에 따라 알림, 방송, 신고로 이어지는 AI 안전 대응 시스템을 구현하는 것이 목표입니다.

프로젝트 개요
  • 문제: CCTV나 영상 속 위험 상황을 사람이 항상 실시간으로 확인하기 어려움
  • 해결: 영상 기반 행동 분석과 위험 물체 탐지를 결합해 위험 상황을 자동 판단
  • 구성: 정상/이상 행동 분류 모델 + YOLO 기반 위험 물체 탐지 + risk_score 계산 로직
  • 목표: 위험 점수에 따라 알림 / 방송 / 신고까지 이어지는 자동 대응 시스템 구현
행동 분석 영상 또는 프레임을 기반으로 정상 행동과 이상 행동을 분류
위험 물체 탐지 YOLO 기반 객체 탐지로 칼과 같은 위험 요소를 감지
위험 점수 계산 행동 결과와 객체 탐지 결과를 조합해 risk_score 산출
자동 대응 위험도에 따라 알림, 방송, 신고 시뮬레이션으로 연결

현재는 모델 구조와 백엔드 분석 흐름을 우선 구현하고 있으며, 추후 실제 분석 화면과 결과 시각화를 추가할 예정입니다.

AI 판단 파이프라인으로 확장 중입니다

단순 객체 탐지에서 끝나는 것이 아니라, 영상 입력 → 행동 분석 → 위험 물체 탐지 → 위험 점수 계산 → 대응 결과까지 이어지는 하나의 AI 판단 시스템 구조로 발전시키고 있습니다.
